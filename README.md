# Advanced SQL Business Analytics

## Project Overview

This project demonstrates advanced SQL analytics using a retail sales data warehouse built on a star schema architecture.

The objective was to derive customer, product, and revenue insights using analytical SQL techniques commonly used in Data Analyst, Business Analyst, and BI Analyst roles.

## Tools & Technologies

* MySQL
* SQL
* Window Functions
* Common Table Expressions (CTEs)
* Data Warehousing
* Star Schema Modeling

## Key Business Questions

* Who are the highest revenue-generating customers?
* Which product categories drive the most revenue?
* How does customer purchasing behavior vary across segments?
* Which regions contribute most to sales performance?

## SQL Concepts Demonstrated

* Joins
* Subqueries
* Aggregate Functions
* CTEs
* Window Functions
* RANK()
* DENSE_RANK()
* ROW_NUMBER()
* Partitioning

## Business Impact

The project simulates real-world business reporting and analytics workflows used by retail, e-commerce, and consulting organizations.
